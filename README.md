# Personalized Webpage using JavaScript

This project is an interactive personalized webpage where users input their name and age. The app stores user data in localStorage, calculates their age in months, shows a personalized greeting, and displays motivational quotes.

## Features
- Persistent storage using localStorage
- Conditional content (adult check)
- Dynamic age calculation
- Motivational quotes displayed in a loop
- Responsive and styled UI

## Tech Stack
- HTML
- CSS (or Tailwind / Bootstrap)
- JavaScript
- Git & GitHub Pages for deployment

## Deployment
1. Push all files to a GitHub repository.
2. Enable GitHub Pages from the repository settings.
3. Access your live site using the provided GitHub Pages URL.
